### Installation
The recommended method for installing ChatterBot is by using `pip`.
### Installing from `PyPi`
```bash
$ sudo pip3 install chatterbot
```
### Installing from source
```bash
$ git clone https://github.com/gunthercox/ChatterBot.git
$ sudo pip3 install ./ChatterBot
```
